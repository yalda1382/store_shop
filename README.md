# store_shop
A simple e-commerce web application inspired by Digikala.
Users can browse products, add items to cart, and place orders.

## Features
- User authentication (login & signup)
- Product listing
- Shopping cart
- Order checkout
- Admin panel (basic)

- ## Technologies Used
- Python
- Django
- HTML / CSS 
-Javascript

## How to Run

1. Clone the repository:
git clone https:(https://github.com/yalda1382/store_shop/)

2. Create virtual environment and activate it :
3. python3 -m venv venv
4. active windows: venv\Scripts\activate
5. active mac & linux :source venv/bin/activate

6. Install dependencies:
pip install -r requirements.txt

7. Run server:
python manage.py runserver

8.localhost:
http://127.0.0.1:8000/
